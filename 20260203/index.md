# 20260203の日記


今日は節分の日。会社のイベントで恵方巻をみんなで恵方を向いて無言で食べるというイベントにお昼に参加した。コンビニの恵方巻が配られたので食したのだが、めちゃくちゃ美味しかった。

値段を今調べてみたら、僕が食べたサラダのは400円ぐらい。ぐっと我慢した焼肉のは900円ぐらいだったので、なんかとても損をした気持ちになってきている。人間とは不思議な生き物である。事実上は、何も損はしていないのに。

こういう構造は、どの世界でも似たようなことが語られる。政治の世界は、お金の配分を決めるのが政治の大切な役割なので、誰かが得した分は自分が損をするという構造になりがちなので特に顕著だ。

誰が得をするとか損をするとかじゃなく、もう少し未来を向いた方向で話し合える世界が来ると良いなと思ってはいるが、今日のこの僕に起きた小さな不満みたいなのを見るにつけ、理想論に過ぎずすごく難しいのだろうなと自戒しているところだ。

2026年2月3日　河添ひろし
