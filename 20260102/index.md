# 20260102の日記


あっという間に元日が終わりまして、今日から動き出します。この後、双子の兄の家に行こうと思ってますので、手短に。

1月2日は書初めの日ですよね。学生時代以降、やることはありませんが、新年の抱負みたいなものを考える日です。

僕は、2025年は本当に激動の1年だったので、今年はちょっとゆっくり目にペースを落としながらも引き続き「挑戦」ということをしていきたいなと思っています。

思い返せば昨年の1月に石丸さんが記者会見で、再生の道の発表を見て応募しようと決意したところから、僕の2025年と人生は大きく動きだしました。

そこからは初めてのことへの挑戦の連続につぐ連続で、2025年上半期のこと思い出そうとしてもあまり思い出せません。思い出せる範囲を細かく書いていっても良いですが、流石に冗長なので、楽しくないことはやらないという精神で活動してたので、最終的な感想は「楽しかった！」というものになります。

2026年がどんな年になるかは楽しみでしかないのですが、新しいことへの挑戦はどんどん続けていきたいと考えています。最近だとAI関連はめちゃくちゃ面白いので、毎週新しい何かを試してみてます。形になるものがあれば、よくわからんというものもありますが、変化の速いIT業界に居ますが今ほど変化が速いと感じたことはありません。一つ上げるとすると、調べものするときにGoogleで検索してサイトを見に行くことなくなりましたよね。アフィリエイトで稼いでた人とか、SEOを売りにしてた人とかちゃんとピボットしないとあっという間に稼げなくなりそうです。

さーて、今年はどんな年になるかなと思いをはせつつ、残りの正月休みを過ごしたいと思います。

2026年1月2日　河添ひろし
