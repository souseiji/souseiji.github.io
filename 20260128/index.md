# 20260128の日記


困った。今日は特に書くことがない。なので、年初から続いてた体調不良について少々書いてみようと思う。

年初といっても、あれ？体調悪いな？と思ったのは、1月15日ぐらいのこと。熱を測っても平熱だけど全身だるいし頭も痛いような気がする。何より眠たくてずっと寝てられる。何か体が疲れているのだろうか？明確なサインは出ていないのが気になるな、というぐらいだった。

本格的に体に不調が出たのはその翌日。1日20時間ぐらい寝てたのに、のどが痛くなり食べ物に困るぐらいだった。流動食としてプリンをここぞとばかり買い込んで、うまうまと食べてたら気づいたらのどの痛みは引いてたのだが、咳が止まらない。これまで咳が出たらなめると治ってた龍角散ののど飴が全く効かなくて、夜中の咳のせいでろくに寝られない日が続いた。ここから先の体調不良は、ほぼほぼ寝不足が原因だ。

咳の症状に困ってたときに、ふと買っておいたけど使ってないうがい薬があることを思い出したので、それを使ってみたらこれが効果てきめんだった。咳も止まりのどの違和感もなくなってきた。が、毎日続けないとすぐに再発するというのが今の現状である。寝るとき含めて、マスクをしておくのも大事だ。

もうすぐ51歳。このぐらいの年齢になると、一つの風邪（？）でも、これだけ治るのに時間がかかるのかとこれから先を考えると頭が痛くなるような事象にでくわした。1晩寝れば治るっしょはもう通用しないし、そんな時代が戻ってくることはなくここから先は悪くなる一方だと考えるとなんとなくノスタルジックな気持ちになってきたあたりで日記を終わろうと思う。
