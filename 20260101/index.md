# 20260101の日記


2026年が始まりました。2025年は、本当にいろんなことに挑戦をしました。

今年は少し落ち着いた年にしたいなと思ってますが、残念ながら世の中の動向的にもそういうことは許されなさそうです。

最近気になってたのは東京都のごみ袋有料化です。西東京市はそもそも有料ごみ袋なので、たぶん無関係です。値上きたらちょっとやだなってぐらいの感じです。

私は5年ぐらい前に練馬区から西東京市に引っ越した時に、ごみ袋有料化でびっくりしたんですが、私が学生時代に過ごした福岡県は10枚800円というめちゃくちゃ高いごみ袋だったので、良心的な価格だなぐらいにしか思いませんでした。

実際にごみ袋が有料になったのをきっかけに、あまりものを買わないという方向にシフトしたのでそこまで痛いという感じはありませんでした。
