# 20260125の日記


日本取引所グループが「内部統制強化・不祥事予防に向けたハンドブック」を出しています。

[内部統制強化・不祥事予防に向けたハンドブック](https://www.jpx.co.jp/regulation/listing/handbook/index.html)

不祥事が発生する原因は紐解いていくと複数の原因が積み重なって不祥事という形で表出することが多いです。その根っこには、企業風土が昔の成功体験のところから更新されてなくて古いままということがあります。この企業風土を変えていくことができるのは会社のトップ、つまり社長以外に変えることはできません。

僕が内部監査という仕事をしてて、一番気を付けているのはここです。他社で発生した不祥事の調査報告書を読みこむのも、自社のベンチマークとして活用するためです。

ただ、当然ながら企業風土だけを注意してればいいわけではなく、最終的には総合力が必要になってきます。そういう視点で、この日本取引所グループのハンドブックはとても役に立ちます。役に立つだけでなく単純に読み物としても面白いので、読み込んでおこうと思います。
